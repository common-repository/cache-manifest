== Description ==

This plugin will cache certain content of your website and will make it available for offline navigation.

If you notice anything wrong, please feel free to use the support forum and describe your scenario.

== Installation ==

1. Upload `cache-manifest.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are done!

== Frequently Asked Questions ==

= How to test this plugin? =

Install it and check your active theme's folder. There you should have two files generated called 'manifes.html' and 'cache.manifest'. Unplug your Internet cable and have fun!

= Can I browse updated content while I'm offline? =

Your latest cached concent is available from the last time you were online and the cache process happened.
